## What
Fetches RSI (Relative Strength Index) values for the top 1000 cryptocurrencies by CoinMarketCap ranking using the Polygon.io API and outputs results as CSV files.

## Why
Automates scanning a large universe of crypto tickers for RSI signals (21-day and 210-day windows) to support technical analysis workflows, working within free-tier API rate limits.
